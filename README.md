# ZeitVibe

Edge AI tools for the KHADAS VIM3 NPU. Calibrating the vibe of the age.

## Projects

### 🛡️ [Guardian](guardian/)
Terminal safety with NPU-powered pattern detection.
- Warns on dangerous commands (`rm -rf`, `sudo rm`)
- Detects `pip install` outside virtual environments
- Git intelligence (warns about commits outside repos)
- Real-time NPU monitoring
- Command history logging

→ [Install & Documentation](guardian/)

### 📓 [Journal](journal/)
Learning edge computing in public. The original VIM3 Journal series.
- NPU journal system
- Install scripts
- Project documentation

→ [Read more](journal/)

### 😍 [Sentiment](sentiment/)
Emoji mood matrix analyzer.
- ONNX runtime inference
- Custom sentiment model
- Emoji bar charts

→ [Install & Documentation](sentiment/)

### 📊 [Trend](trend/)
SBC trend detector for the maker community.
- Weekly trend reports
- Mention counting across subreddits
- Growth analysis

→ [Documentation](trend/)
